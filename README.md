# master-layer-ai

Simple repo for my local llm stuff, which is currently just a docker instance running Ollama and Open WebUI

### Usage for `run.sh`
```bash
# Start containers
./run.sh up

# Stop containers
./run.sh down

# Restart containers
./run.sh restart

# View logs (live)
./run.sh logs

# Check container status
./run.sh status
```
